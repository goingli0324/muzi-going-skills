# 多角度組隊思考 (Multi-Perspective Panel)

## 核心理念

每個人都有認知盲點,單一視角看決策容易漏掉重要面向。這個 skill 讓你同時從四個彼此獨立、幾乎不重疊的角度檢視同一個問題,適用於任何決策領域——教學決策、專案規劃、博論的情境分析,或任何你正在想的事。

## 四個角色與各自的思考工具

四個角色對應批判思考、創意思考、有效溝通、有效互動四種核心能力,彼此職責刻意不重疊:

| 角色 | 關注面向 | 具體任務 |
|---|---|---|
| **批判思考者** | 評估主張、分析推論、權衡決策、分析問題 | 找出邏輯漏洞、未驗證的假設、可能的偏誤,質疑證據是否充分 |
| **創意思考者** | 促進發現、解決問題、創造新方案/流程/服務 | 提出替代方案、跨領域類比、把限制轉化為機會 |
| **溝通思考者** | 有效運用語言、有效運用非語言溝通 | 評估怎麼對不同對象說明,找出論述結構與表達上的弱點 |
| **互動思考者** | 協商調解說服、與他人有效合作、解決倫理難題、社會意識 | 分析利害關係人反應、連鎖效應、潛在的倫理問題與權力關係 |

## 判斷邏輯:選平行分析還是紅藍辯論

判斷標準只看**使用者句子裡出現哪一類決策詞**,不憑感覺猜「聽起來定案了沒」:

| 句中出現的信號詞 | 走哪個模式 |
|---|---|
| 疑問/探索詞:「要不要」「該不該」「還在想」「不知道怎麼選」「有哪些可能」 | **模式 1:四角色平行分析** |
| 完成式決策詞:「已經決定」「打算」「準備要」「下週就要」「我的方案是」 | **模式 2:紅藍隊辯論** |
| 兩類詞都出現(例如「我打算 X,但還在想要不要」) | 以疑問詞優先 → **模式 1**(還沒真正定案) |
| 兩類詞都沒有 | 才問使用者一句:「你是還在想方向,還是已經有初步決定要我幫你測試?」 |

### 模式 1:四角色平行分析

四個角色各自獨立發表 150-250 字左右的分析,不互相對話、不對抗,最後由你統整成一份「綜合觀察」,點出四個角度中彼此呼應或衝突的地方。

### 模式 2:紅藍隊辯論

- **紅隊(挑戰方)**:由批判思考者+互動思考者組成,全力找風險、漏洞、潛在災難,語氣不留情面
- **藍隊(辯護方)**:由創意思考者+溝通思考者組成,針對紅隊每個質疑提出回應或修正方案,不是盲目辯護
- **流程**:紅隊先攻(3-5 個核心質疑)→ 藍隊逐一回應 → 紅隊針對薄弱回應追問 → 你以裁判身分做最終裁定:哪些質疑是致命的必須修正、哪些是過度擔心可以接受、最終建議是「執行/修改後執行/放棄」

## 博論 CLA 延伸(僅在使用者明確處於論文情境時使用)

若使用者的問題明顯屬於博論的情境規劃或 CLA 分析(例如提到臺灣華語教育產業未來情境),完成四角色分析或辯論後,可以主動提議:「要不要我把這四個角度的觀察,對應到 CLA 的四個層次(表象層/系統層/世界觀層/迷思隱喻層)整理一次?」——但不要在單純的日常教學決策上主動套 CLA 術語,以免小題大作。

## 使用範例(示意)

**使用者輸入**:「我在想要不要把 AI 數位增能模組從選修改成必修,還沒決定。」

**判斷**:開放問題,沒有定案 → 模式 1(四角色平行分析)。

**使用者輸入**:「我已經決定把 AI 數位增能模組改成必修,下學期就這樣排,你幫我看看有沒有問題。」

**判斷**:已有具體決定 → 模式 2(紅藍隊辯論)。
